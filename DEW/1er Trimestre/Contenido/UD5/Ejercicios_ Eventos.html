El gato caminante

  • Completa esta Web que contiene un tab img de un gato caminando, y funciones JavaScript que harán que el gato se mueva por la pantalla. 
    <!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8" />
  <title>El gato caminante</title>
 </head>
 <body>
  
  <button id="start-button">Start</button>
  <button id="speed-button">Go Faster!</button>
  <button id="stop-button">Stop</button>
  <div id="info"></div>
  
  <img style="position:absolute; left: 0px; top: 80px;" src="http://www.anniemation.com/clip_art/images/cat-walk.gif">

  <script>
var movePixels = 10;
var delayMs = 50;
var catTimer = null;
function catWalk() {
  var img = document.getElementsByTagName('img')[0];
  var currentLeft = parseInt(img.style.left);
  img.style.left = (currentLeft + movePixels) + 'px';
  if (currentLeft > (window.innerWidth-img.width)) {
    img.style.left = '0px';
  }
}
function startCatWalk() {
  catTimer = window.setInterval(catWalk, delayMs);
}
  </script>

 </body>
</html>
  • (No necesitas modificar el código original - añade el código a continuación.)
  • Añade un event listener al botón start, de manera que el gato comience a moverse por la pantalla cuando haces click.
  • Añade un event listener al botón stop, de manera que el gato se detenga cuando haces click.
  • Añade un event listener al botón speed, de manera que el gato se mueve más rápido cuando haces click. 



    // #1
function onStartClick() {
    startCatWalk();
}
var startButton = document.getElementById('start-button');
startButton.addEventListener('click', onStartClick);

// #2
function onStopClick() {
   window.clearInterval(catTimer);
}
var stopButton = document.getElementById('stop-button');
stopButton.addEventListener('click', onStopClick);

// #3
function onSpeedClick() {
    movePixels += 5;
    var speed = movePixels * (1000/50);
    document.getElementById('info').innerHTML = 'Current speed: ' + speed + 'px/second';
}
var speedButton = document.getElementById('speed-button');
speedButton.addEventListener('click', onSpeedClick);
